# 🚀 Git Workshop

Welcome to the Hands-On Git Workshop! 
This repository simulates a standard Data Science project environment. We have a raw dataset and a Python script designed to clean it. 

Your mission today is to collaborate on this pipeline without breaking the main codebase.

## 📋 Prerequisites
Before we begin, ensure you have:
1. Created a GitHub account.
2. Installed [Git](https://git-scm.com/downloads) on your machine.
3. Installed a code editor (we will be using VS Code today).

## 🛠️ The Workshop Workflow

### Step 1: Get Your Own Copy
1. Click the **Fork** button at the top right of this page to create a copy of this repository under your own account.
